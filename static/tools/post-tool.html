⚙️ GitHub トークン設定

GitHub → Settings → Developer settings → Personal access tokens (Fine-grained) で発行。
権限：Contents: Read and write のみ必要。

✅ トークンが保存されています
📋 .md ファイル内容 フロントマター＋本文をそのまま貼り付け

記事化プロンプトの「■ .mdファイル内容（フロントマター＋本文HTML）」をそのまま貼り付けてください。

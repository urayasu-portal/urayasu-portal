<p>赤ちゃんが生まれてから就学前までの乳幼児期には、節目の健康診査や予防接種、親子で過ごせる場所、ちょっと預けたいときの支援など、浦安市のさまざまなサポートがあります。このページは「自分に関係しそうな支援に気づき、正しい窓口にたどり着く」ための地図です。<strong>対象年齢・料金・申込み方法などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>困ったときの入口は2つ</strong><br>
・健診・予防接種・育児の相談 → <strong>母子保健課（健康センター1階）</strong>　電話：047-381-9034<br>
・親子の居場所・身近な相談 → <strong>子育て支援センター／こどもの広場</strong>（市内各所）<br>
健診や予防接種は対象月齢になると個別に案内が届きます。届いた通知をなくさないように保管しておきましょう。
{{< /note >}}

<h2>① 節目で受ける健康診査</h2>

<p>乳幼児健診は対象の月齢・年齢になると個別に通知が届きます（公費で受けられます）。発育・発達の確認のほか、育児の相談もできる機会です。</p>

<h3>乳児健康診査</h3>
<p>生後まもない時期の乳児を対象とした健康診査です。対象月齢になると案内が届きます。</p>
<p><strong>対象：</strong>浦安市に住民票がある乳児（対象月齢は通知でご確認ください）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/1000793.html" >}}乳児健康診査（浦安市公式）{{< /gov-link >}}

<h3>1歳6か月児健康診査</h3>
<p>1歳6か月ごろの発育・発達やことば、歯の状態などを確認する健診です。</p>
<p><strong>対象：</strong>1歳6か月を過ぎた市内在住のお子さん（個別に通知）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/1000781.html" >}}1歳6か月児健康診査（浦安市公式）{{< /gov-link >}}

<h3>3歳児健康診査</h3>
<p>3歳ごろの発育・発達、視覚・聴覚などを確認する健診です。</p>
<p><strong>対象：</strong>3歳5か月を過ぎた市内在住のお子さん（個別に通知）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/1000782.html" >}}3歳児健康診査（浦安市公式）{{< /gov-link >}}

<h2>② 予防接種</h2>

<h3>子どもの定期予防接種</h3>
<p>法令で定められた種類・回数・間隔にそって、対象年齢の間は公費（無料）で受けられます。接種には母子健康手帳が必要です。スケジュールは市の案内や「浦安市おやこ手帳」で確認できます。</p>
<p><strong>対象：</strong>各ワクチンの対象年齢にある市内在住のお子さん</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/yobou/index.html" >}}子どもの予防接種（浦安市公式）{{< /gov-link >}}

<h3>子どものインフルエンザ予防接種費用の助成（任意接種）</h3>
<p>任意接種であるインフルエンザ予防接種の費用の一部を助成します（1回あたり上限2,000円。2026年6月時点。例年10月開始で年度ごとに実施）。申込書は市内の協力医療機関にあります。</p>
<p><strong>対象：</strong>接種日に浦安市に住民票がある生後6か月〜高校3年生相当の方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/yobou/index.html" >}}子どもの予防接種・インフルエンザ費用助成（浦安市公式）{{< /gov-link >}}

<h2>③ 親子で通える・相談できる場</h2>

<h3>子育て支援センター</h3>
<p>就学前の子どもと保護者が気軽に集い、遊んだり交流したりできる場です。育児の相談や情報提供、小児科医による相談などもあります。</p>
<p><strong>対象：</strong>市内の未就学児とその保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/shisetsu/kosodate/kosodate/1005523.html" >}}子育て支援センター（浦安市公式）{{< /gov-link >}}

<h3>こどもの広場</h3>
<p>親子で自由に遊べる身近な居場所です。近所で気軽に立ち寄れます。</p>
<p><strong>対象：</strong>市内の乳幼児とその保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/shisetsu/kosodate/kosodate/1011997.html" >}}こどもの広場（浦安市公式）{{< /gov-link >}}

<h3>離乳食ステップアップクラス</h3>
<p>離乳食の進め方を学べる教室です。月齢に合わせた進め方の不安を相談できます。</p>
<p><strong>対象：</strong>市内の乳児の保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/1009019.html" >}}離乳食ステップアップクラス（浦安市公式）{{< /gov-link >}}

<h3>子育て相談（育児の不安・悩み）</h3>
<p>育児の不安や悩みを相談できる窓口です。どこに相談すればよいか分からないときの入口にもなります。</p>
<p><strong>対象：</strong>市内で子育てをしている方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/joho/1006623.html" >}}子育て相談（浦安市公式）{{< /gov-link >}}

<h2>④ 預ける・ちょっと手伝ってほしいとき</h2>

<p>仕事や用事、リフレッシュ、急な事情などで一時的に子どもを預けたいときの支援です。</p>

{{< warn >}}一時預かりやファミリー・サポート・センターの利用料は、施設・形態・利用時間によって異なります。また事前の登録や予約が必要な場合があります。利用を予定している施設・事業のページで必ずご確認ください。{{< /warn >}}

<h3>理由を問わない一時預かり（半日・1日利用）</h3>
<p>理由を問わず、半日または1日単位で子どもを預けられる事業です。</p>
<p><strong>対象：</strong>市内在住の乳幼児（施設ごとの条件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/hoiku/1000818.html" >}}理由を問わない一時預かり（浦安市公式）{{< /gov-link >}}

<h3>一時預かり（非定型・緊急・私的事由）</h3>
<p>就労・就学・出産・病気・介護などで家庭での保育が難しいとき、保育園・認定こども園などで一時的に預けられる事業です。</p>
<p><strong>対象：</strong>家庭での保育が一時的に難しい市内在住の乳幼児</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/hoiku/1000813.html" >}}一時預かり（浦安市公式）{{< /gov-link >}}

<h3>うらやすファミリー・サポート・センター</h3>
<p>子どもの送迎や預かりなどを、地域で「お願いしたい人」と「手伝える人」が会員同士で助け合う仕組みです。利用には事前の会員登録が必要です。</p>
<p><strong>対象：</strong>市内在住・在勤などで子育ての援助を受けたい方（会員登録制）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/hoiku/1000815.html" >}}うらやすファミリー・サポート・センター（浦安市公式）{{< /gov-link >}}

<h2>⑤ 発達やことばが気になるとき</h2>

<h3>こども発達センター</h3>
<p>発達の遅れや育てにくさが気になるとき、発達相談・個別療育・通所支援などを専門スタッフが行う施設です（総合福祉センター内）。</p>
<p><strong>対象：</strong>発達に心配のある市内のお子さんとその家族</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/shisetsu/kosodate/kosodate/1005525.html" >}}こども発達センター（浦安市公式）{{< /gov-link >}}

<h3>こどもの発達などの相談</h3>
<p>ことばや行動、発育・発達などで気になることを相談できる窓口です。まずどこに相談すればよいか迷うときの入口になります。</p>
<p><strong>対象：</strong>発達などに心配のある市内のお子さんの保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/soudan/1006621.html" >}}こどもの発達などの相談（浦安市公式）{{< /gov-link >}}

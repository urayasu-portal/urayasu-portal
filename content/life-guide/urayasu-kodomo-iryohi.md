<p>浦安市には、子どもの医療費の自己負担を助成する「子ども医療費助成（マル子）」をはじめ、治療が必要なときの医療費支援や、急な発熱・けがのときの相談窓口があります。このページは「自分に関係しそうな支援に気づき、正しい窓口にたどり着く」ための地図です。<strong>対象年齢・所得要件・助成内容などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>まずは「子ども医療費助成（マル子）」の申請を</strong><br>
出生届・転入のあと、子育て支援課で受給券（マル子）を申請しておくと、ふだんの通院・入院の自己負担を助成してもらえます。受給券は早めに準備しておきましょう。
{{< /note >}}

<h2>① 子どもの医療費助成（マル子）</h2>

<h3>子ども医療費助成（マル子）</h3>
<p>子どもが医療機関にかかったときの保険診療の自己負担分を助成する制度です。受給券と保険証を千葉県内の医療機関・薬局で提示すると、窓口での支払いが原則かかりません（県外を受診した場合などは、いったん支払い後に申請して払い戻す「償還払い」になります）。対象は0歳から高校生相当年齢（18歳に達する日以後の最初の3月31日）までです（2026年6月時点）。</p>
<p><strong>対象：</strong>浦安市に住民票がある0歳〜高校生相当年齢の子ども</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/teate/1000779.html" >}}子ども医療費助成（浦安市公式）{{< /gov-link >}}

<h2>② 治療・手術が必要なときの医療費支援</h2>

<h3>自立支援医療（育成医療）</h3>
<p>体に障がいのある18歳未満の子どもが、手術などの治療で機能の改善が見込まれる場合に、その治療にかかる医療費を支援する制度です。</p>
<p><strong>対象：</strong>身体に障がいがあり、治療で改善が見込まれる18歳未満の子ども（要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/shogai/enjo/iryo/1015736.html" >}}自立支援医療（育成医療）（浦安市公式）{{< /gov-link >}}

{{< note >}}
未熟児の入院治療にかかる「未熟児養育医療」は<a href="/life-guide/urayasu-ninshin-shussan-shien/">妊娠・出産時の支援ガイド</a>でまとめています。
{{< /note >}}

<h2>③ 急な発熱・けがで受診を迷ったとき</h2>

<h3>こども急病電話相談（#8000）</h3>
<p>夜間や休日に子どもが急に熱を出したときなど、受診したほうがよいか迷ったら、看護師などに電話で相談できます。短縮ダイヤル「#8000」のほか、市の救急ページに最新の相談先・急病診療所の情報がまとまっています。</p>
<p><strong>対象：</strong>子どもの急な発熱・けがなどで受診を迷っている方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/todokede/anzen/kyukyu/1038262.html" >}}医療健康相談・急病診療所（浦安市公式）{{< /gov-link >}}

{{< note >}}
<strong>休日・夜間の診療所・当番医をくわしく知りたいとき</strong><br>
小児・大人の休日夜間診療や当番医のまとめは、<a href="/life-guide/holiday-night-medical/">浦安市の休日・夜間診療、急病時の相談先まとめ</a>をご覧ください。
{{< /note >}}

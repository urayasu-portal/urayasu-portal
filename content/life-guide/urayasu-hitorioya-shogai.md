<p>ひとり親家庭や、障がい・発達に心配のある子どもを育てる家庭には、手当・医療費の助成、療育・相談、学校での特別支援など、浦安市のさまざまな支援があります。このページは「自分（お子さん）に関係しそうな支援に気づき、正しい窓口にたどり着く」ための地図です。<strong>手当の額・所得要件・対象などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>相談の入口</strong><br>
・ひとり親家庭の手当・くらしの相談 → <strong>子育て支援課</strong><br>
・障がい・発達に関する相談 → <strong>こども発達センター</strong>、障がい福祉課<br>
どこに相談すればよいか迷うときも、まずは上の窓口へお問い合わせください。
{{< /note >}}

<h2>① ひとり親家庭の手当・医療費</h2>

<h3>児童扶養手当</h3>
<p>離婚・死別・未婚などでひとり親家庭になった方が、子ども（原則18歳年度末まで）を養育している場合に支給される手当です。</p>
{{< warn >}}手当の月額・所得制限・支給月などは、制度改正や世帯の状況によって変わります。金額は本文では断定できないため、必ず公式ページでご自身の場合をご確認ください。{{< /warn >}}
<p><strong>対象：</strong>ひとり親家庭などで18歳年度末までの子どもを養育している方（所得制限あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/teate/1000822.html" >}}児童扶養手当（浦安市公式）{{< /gov-link >}}

<h3>ひとり親家庭等医療費等助成</h3>
<p>ひとり親家庭の親と子どもが医療機関にかかったときの自己負担分を助成する制度です（所得制限あり）。子ども医療費助成とあわせて活用できます。</p>
<p><strong>対象：</strong>市内のひとり親家庭の親と子ども（所得制限あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/teate/1000825.html" >}}ひとり親家庭等医療費等助成（浦安市公式）{{< /gov-link >}}

<h2>② ひとり親家庭の暮らし・自立の支援</h2>

<h3>ひとり親家庭の福祉サービス（資金貸付・住宅手当・就業支援など）</h3>
<p>母子・父子・寡婦福祉資金の貸付、ひとり親家庭住宅手当、資格取得などの自立支援、ホームヘルプサービスなど、暮らしと自立を支えるサービスがあります。手当・助成のまとめページから各制度を確認できます。</p>
<p><strong>対象：</strong>市内のひとり親家庭（各制度に要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/teate/index.html" >}}手当・助成（ひとり親家庭向け含む）（浦安市公式）{{< /gov-link >}}

<h2>③ 障がいのある子どもへの手当</h2>

<h3>障がいのある子どもへの手当（特別児童扶養手当など）</h3>
<p>在宅で障がいのある子どもを養育している家庭への手当には、特別児童扶養手当（国）・障害児福祉手当（国）・心身障がい児手当（市）などがあります。障がいの程度や所得などの要件があります。</p>
<p><strong>対象：</strong>在宅で障がいのある子どもを養育している方（障がいの程度・所得などの要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/shogai/teate/index.html" >}}障がい者への手当（浦安市公式）{{< /gov-link >}}
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/shogai/teate/1001223.html" >}}特別児童扶養手当（浦安市公式）{{< /gov-link >}}

<h2>④ 障がいのある子どもの相談・療育・通所支援</h2>

<h3>こども発達センター（発達相談・療育）</h3>
<p>発達の遅れや育てにくさが気になるとき、発達相談・個別療育・通所支援などを専門スタッフが行う施設です（総合福祉センター内）。</p>
<p><strong>対象：</strong>発達に心配のある市内のお子さんとその家族</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/shisetsu/kosodate/kosodate/1005525.html" >}}こども発達センター（浦安市公式）{{< /gov-link >}}

<h3>障害児通所支援（児童発達支援・放課後等デイサービス）</h3>
<p>障がいのある子どもが、児童発達支援（未就学）や放課後等デイサービス（就学児）などの事業所に通って支援を受けるための給付です。利用には申請・受給者証が必要です。</p>
<p><strong>対象：</strong>支援が必要な市内の障がい児（要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/shogai/seido/1037113.html" >}}障害児通所支援の利用について（浦安市公式）{{< /gov-link >}}

<h3>療育手帳</h3>
<p>知的障がいのある方が、各種の福祉サービスや手当を受けるために必要となる手帳です。</p>
<p><strong>対象：</strong>知的障がいのある方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/shogai/techo/1001219.html" >}}療育手帳（浦安市公式）{{< /gov-link >}}

<h2>⑤ 学校での特別支援教育</h2>

<h3>特別支援学級・通級指導教室（就学相談）</h3>
<p>特別な支援が必要な子どものための特別支援学級や、ことば・きこえ・学びなどに応じた通級指導教室があります。就学・進学に向けた相談は教育センター「まなびサポート」で受けられます。</p>
<p><strong>対象：</strong>特別な教育的支援が必要な市内の児童生徒とその保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/shien/1000941.html" >}}特別支援学級・通級指導教室（浦安市公式）{{< /gov-link >}}

<p>浦安市には、認可保育園・認定こども園・幼稚園・家庭的保育（保育ママ）など、子どもを預けながら育てる選択肢が複数あります。このページは「自分に合う預け先・支援に気づき、正しい窓口にたどり着く」ための地図です。<strong>入園の要件・保育料・申込み時期などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>申込み・保育料の窓口は保育幼稚園課（市役所）</strong><br>
認可保育園・認定こども園の翌年4月入園の一次申込みは、例年秋ごろ（10〜11月）に始まります。年度や園により日程が異なるため、最新の募集時期は必ず公式ページで確認してください。
{{< /note >}}

<h2>① 認可保育園・認定こども園に申し込む</h2>

<h3>認可保育園・認定こども園とは（園を探す）</h3>
<p>就労などで家庭での保育が難しい家庭のための施設です。市内の園は一覧から探せます。</p>
<p><strong>対象：</strong>保育の必要性が認められる家庭の0歳〜就学前の子ども</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/index.html" >}}認可保育園・認定こども園（浦安市公式）{{< /gov-link >}}
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1000867.html" >}}認可保育園一覧（浦安市公式）{{< /gov-link >}}

<h3>入園のための要件（保育の必要性）</h3>
<p>就労・妊娠・出産・病気・介護・求職などの「保育の必要性」があるかどうかで、申込みの可否や認定区分が決まります。</p>
<p><strong>対象：</strong>認可保育施設への入園を希望する家庭</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1033663/1033679.html" >}}入園のための要件（浦安市公式）{{< /gov-link >}}

<h3>入所・転園の申し込み</h3>
<p>申込みの流れ・必要書類・受付時期をまとめたページです。入所の可否は「保育の必要性」の認定と利用調整（選考）で決まります。はじめての方は入所ガイドブックも参考になります。</p>
<p><strong>対象：</strong>認可保育施設への入園・転園を希望する家庭</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1033663/index.html" >}}入所・転園の申し込み（浦安市公式）{{< /gov-link >}}
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1033663/1043588.html" >}}保育園・認定こども園入所ガイドブック（浦安市公式）{{< /gov-link >}}

<h2>② 保育料・給食費・無償化</h2>

<h3>幼児教育・保育の無償化</h3>
<p>3〜5歳児クラスの幼稚園・保育園・認定こども園などの利用料は無償化されています（副食費・教材費・延長保育料などは対象外）。私立幼稚園や認可外保育施設も一定の上限まで対象です。上限額や対象範囲は変わることがあるため、公式ページでご確認ください。</p>
<p><strong>対象：</strong>3〜5歳児クラスの子ども（住民税非課税世帯は0〜2歳児も対象）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/1026268/index.html" >}}幼児教育・保育の無償化（浦安市公式）{{< /gov-link >}}

<h3>0〜2歳児クラスの保育料無償化</h3>
<p>浦安市では、令和8年（2026年）4月1日から、認可保育施設などを利用する0〜2歳児クラスの保育料を無償化しました（2026年6月時点。給食費などは別途必要）。</p>
<p><strong>対象：</strong>認可保育施設などを利用する0〜2歳児クラスの子ども</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1047820.html" >}}0〜2歳児クラスの保育料無償化（浦安市公式）{{< /gov-link >}}

<h3>保育料・給食費</h3>
<p>保育料の決まり方（世帯の市民税額による区分）や給食費、支払い方法をまとめたページです。</p>
{{< warn >}}保育料・給食費は、世帯の所得（市民税額）・子どもの年齢・きょうだいの人数・利用する施設などによって異なります。金額は本文では断定できないため、必ず公式ページでご自身の区分をご確認ください。{{< /warn >}}
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1042934/index.html" >}}保育料・給食費（浦安市公式）{{< /gov-link >}}

<h2>③ 幼稚園・認定こども園（教育を中心に）</h2>

<h3>幼稚園・認定こども園とは</h3>
<p>3歳から就学前までの子どもが教育を受ける場です。市立・私立があり、預かり保育を実施している園もあります。</p>
<p><strong>対象：</strong>3歳〜就学前の子ども</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/youchien/index.html" >}}幼稚園・認定こども園（浦安市公式）{{< /gov-link >}}

<h3>市立幼稚園・認定こども園の一覧・園紹介</h3>
<p>市立の幼稚園・認定こども園の一覧と、各園の紹介ページです。</p>
<p><strong>対象：</strong>市立の幼稚園・認定こども園を検討している家庭</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/youchien/1043055/index.html" >}}市立幼稚園・認定こども園一覧（浦安市公式）{{< /gov-link >}}

<h2>④ いろいろな預け方・困ったときの預かり</h2>

<h3>家庭的保育（保育ママ）</h3>
<p>保育士などの資格を持つ「保育ママ」が、少人数の子どもを家庭的な雰囲気で預かる事業です。</p>
<p><strong>対象：</strong>保育の必要性が認められる生後6か月〜3歳未満の子ども</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/hoiku/hoikuen/1000850.html" >}}家庭的保育（保育ママ）事業（浦安市公式）{{< /gov-link >}}

<h3>病児・病後児保育</h3>
<p>病気中・回復期の子どもを、仕事などで家庭で看られないときに預けられる事業です。市内の医療機関併設施設などで利用できます（事前登録が必要）。</p>
<p><strong>対象：</strong>病気中・回復期で集団保育が難しい市内の子ども</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/hoiku/1000810.html" >}}病児・病後児保育（浦安市公式）{{< /gov-link >}}

<h3>訪問型病児・病後児保育の利用料補助</h3>
<p>ベビーシッター等による訪問型の病児・病後児保育を利用した場合に、利用料の一部を補助します。</p>
<p><strong>対象：</strong>訪問型の病児・病後児保育を利用した市内在住の家庭（要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/hoiku/1032552.html" >}}訪問型病児・病後児保育の補助（浦安市公式）{{< /gov-link >}}

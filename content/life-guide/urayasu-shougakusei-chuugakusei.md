<p>小学校・中学校に上がると、放課後の過ごし方や、学校生活・いじめ・不登校など、相談したいことが変わってきます。このページは「自分（お子さん）に合う支援・相談先に気づき、正しい窓口にたどり着く」ための地図です。<strong>対象・申込み時期・利用料などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>悩んだときの相談先は2つの入口</strong><br>
・学校生活・いじめ・友だち・不登校の悩み → 各学校の<strong>スクールライフカウンセラー</strong>、または<strong>教育センター（いちょう学級）</strong><br>
・放課後の居場所 → <strong>児童育成クラブ（学童）／放課後子ども教室</strong><br>
ひとりで抱え込まず、早めに相談してください。
{{< /note >}}

<h2>① 放課後の過ごし方・居場所</h2>

<h3>児童育成クラブ（学童保育）</h3>
<p>保護者が就労などで昼間家庭にいない小学生を対象に、放課後や長期休業中の生活・遊びの場を提供します。4月入会の申込みは前年の秋ごろ（11〜12月）が中心です。</p>
{{< warn >}}保護者負担金（利用料）や開所時間・対象学年などは年度や状況により異なります。金額は本文では断定できないため、最新の入会案内（公式ページ）でご確認ください。{{< /warn >}}
<p><strong>対象：</strong>保護者が就労などで昼間家庭にいない市内在住の小学生</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/1011200/1000934.html" >}}児童育成クラブ（浦安市公式）{{< /gov-link >}}

<h3>放課後子ども教室</h3>
<p>学校施設などを使い、子どもが自由に参加して遊べる放課後の居場所です。出入りは自由で、児童育成クラブとは別のしくみです。</p>
<p><strong>対象：</strong>市内の小学生</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/1011200/1001035.html" >}}放課後子ども教室（浦安市公式）{{< /gov-link >}}

<h2>② 学校生活・いじめ・友だちの悩み</h2>

<h3>いじめ110番</h3>
<p>いじめで困っているとき、子ども本人・保護者が相談できる窓口です。各学校に配置されたスクールライフカウンセラーにも相談できます。</p>
<p><strong>対象：</strong>いじめや学校生活で困っている子ども・保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/todokede/soudan/kyoiku/1000704.html" >}}いじめ110番（浦安市公式）{{< /gov-link >}}

<h3>地域こども相談</h3>
<p>身近な地域で、子どもや子育てに関する悩みを相談できる窓口です。どこに相談すればよいか迷うときの入口にもなります。</p>
<p><strong>対象：</strong>市内の子ども・保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/todokede/soudan/kyoiku/1040281.html" >}}地域こども相談（浦安市公式）{{< /gov-link >}}

<h2>③ 学校に行きづらい・不登校のとき</h2>

<h3>不登校支援・教育相談（教育センター「いちょう学級」）</h3>
<p>学校に行きづらい子どもや保護者に、心理師・教員経験者などが教育相談・学習支援・家庭訪問相談などを行います。学校以外の居場所・学びの場として利用できます。</p>
<p><strong>対象：</strong>学校に行きづらい市内の小・中学生とその保護者</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/tetsuduki/1000932.html" >}}不登校支援・教育相談（浦安市公式）{{< /gov-link >}}

<h3>学びの多様化学校（分教室）</h3>
<p>不登校の子どもに合わせた柔軟な教育を行う学校（分教室）です。一人ひとりの状況に応じた学びができます。</p>
<p><strong>対象：</strong>不登校などで柔軟な学びを必要とする市内の児童生徒</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/shokai/1043661.html" >}}学びの多様化学校（分教室）（浦安市公式）{{< /gov-link >}}

<h2>④ 学習・生活のサポート</h2>

<h3>子どもの学習・生活支援事業</h3>
<p>学習の仕方や学校の勉強、学校・家庭以外の居場所が必要な子どもへの学習支援・居場所づくりを行う事業です。保護者の相談にも応じます。</p>
<p><strong>対象：</strong>支援を必要とする市内の子ども・家庭（要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/seikatsu/shien/1041870.html" >}}子どもの学習・生活支援事業（浦安市公式）{{< /gov-link >}}

{{< note >}}
・学用品費・給食費などの就学援助や奨学支援金は<a href="/life-guide/urayasu-jido-teate/">児童手当・経済的支援ガイド</a>へ。<br>
・雨の日にも使える屋内の遊び場・児童センターは<a href="/life-guide/indoor-playgrounds/">浦安市の子ども向け屋内・公共の遊び場まとめ</a>へ。<br>
・発達や特別支援教育に関する支援は、ひとり親・障がいのある子どもへの支援ページ（準備中）でまとめる予定です。
{{< /note >}}

<p>浦安市では、妊娠から出産・産後まで、健診費用の助成や給付金、産後ケアなど多くの支援があります。このページは「自分に関係しそうな支援に気づき、正しい窓口にたどり着く」ための地図です。<strong>金額・期限・対象などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>まずは母子保健課へ（健康センター1階）</strong><br>
妊娠届を出すときの面談で、母子健康手帳・各種受診票の交付、妊婦支援給付金の案内、産後ケアの相談までまとめて受けられます。多くの支援はここが入口です。<br>
電話：047-381-9034 ／ 047-381-9058
{{< /note >}}

<h2>① 妊娠がわかったら</h2>

<h3>妊娠届・母子健康手帳の交付</h3>
<p>妊娠したら届け出て、母子手帳と各種受診票を受け取る最初の手続きです。</p>
<p><strong>対象：</strong>浦安市に住民票がある妊娠した方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/ninpu/1000778.html" >}}母子健康手帳について（浦安市公式）{{< /gov-link >}}

<h3>妊婦のための支援給付（妊婦支援給付金）</h3>
<p>妊娠届出時と妊娠後期の2回に分けて給付金が支給されます。1回目は妊婦1人あたり5万円、2回目は子ども1人あたり5万円です（2026年6月時点。申請期限・要件は公式でご確認ください）。</p>
<p><strong>対象：</strong>令和7年4月1日以降に妊娠した方など</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/1039002.html" >}}妊婦のための支援給付（浦安市公式）{{< /gov-link >}}

<h2>② 妊娠中の健診・教室</h2>

<h3>妊婦健康診査の助成</h3>
<p>受診票を使って、妊娠中の健診費用を公費で助成します（合計14回まで・一部自己負担あり。2026年6月時点）。多胎妊娠の方は追加助成があります。</p>
<p><strong>対象：</strong>浦安市に住民票がある妊婦の方（所得制限なし）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/ninpu/1000775.html" >}}妊婦健康診査（浦安市公式）{{< /gov-link >}}

<h3>妊婦歯科健康診査</h3>
<p>妊娠中に受診券で歯科健診を受けられます。</p>
<p><strong>対象：</strong>浦安市に住民票がある妊婦の方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/ninpu/1029164.html" >}}妊婦歯科健康診査（浦安市公式）{{< /gov-link >}}

<h3>初回産科受診料の助成（低所得の方向け）</h3>
<p>住民税非課税世帯など低所得の方が、妊娠判定の初回受診料の助成を受けられます。</p>
<p><strong>対象：</strong>低所得で妊娠判定を受ける方（所得などの要件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/ninpu/1039144.html" >}}初回産科受診料の助成（浦安市公式）{{< /gov-link >}}

<h3>麻しん風しん（MR）予防接種費用の助成</h3>
<p>妊婦本人は接種できないため、妊娠を希望する方やそのパートナーなどへの接種費用を助成します（自己負担あり・接種前の事前申請が必要）。</p>
<p><strong>対象：</strong>浦安市に住民票がある18歳以上で、妊娠を予定・希望する女性とそのパートナー、妊娠中の方のパートナーなど</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/yobou/josei/1001142.html" >}}麻しん風しん予防接種の助成（浦安市公式）{{< /gov-link >}}

<h3>ウエルカム！ベイビークラス（両親学級）</h3>
<p>妊娠・出産・産後の育児を学ぶ教室です（妊娠・出産編／産後・育児編）。</p>
<p><strong>対象：</strong>浦安市の妊婦の方とそのパートナー</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/ninpu/1000777.html" >}}ウエルカム！ベイビークラス（浦安市公式）{{< /gov-link >}}

<h3>妊娠中の相談・訪問</h3>
<p>保健師・助産師による相談支援があります。子育てケアプランの作成、産前・産後サポーターの訪問、妊婦訪問など、不安なときに相談できる窓口です。</p>
<p><strong>対象：</strong>浦安市の妊婦の方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/shien/1010290.html" >}}子育てケアプラン（浦安市公式）{{< /gov-link >}}

<h2>③ 出産前後のお金の手続き</h2>

<h3>出産育児一時金</h3>
<p>出産したときに支給される一時金です。浦安市の国民健康保険に加入している方は1児あたり50万円です（2026年6月時点）。他の健康保険に加入の方は、加入先の保険者へお問い合わせください。</p>
<p><strong>対象：</strong>浦安市国民健康保険に加入している方など</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/kokuho/kyufu/1001312.html" >}}出産育児一時金（浦安市公式）{{< /gov-link >}}

<h3>産前産後期間の国民健康保険税の減額</h3>
<p>出産前後の一定期間、国保税の一部が減額されます（対象期間は単胎で4か月・多胎で6か月分。2026年6月時点）。</p>
<p><strong>対象：</strong>浦安市国保に加入し出産する（した）方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/fukushi/kokuho/iryohi/1041391.html" >}}産前産後期間の国保税減額（浦安市公式）{{< /gov-link >}}

<h2>④ 産後のケア・健診・訪問</h2>

<h3>産婦健康診査・1か月児健康診査</h3>
<p>出産後まもない時期の産婦健診と、1か月児健診の費用の一部を助成します。</p>
<p><strong>対象：</strong>受診日に浦安市に住民票がある産婦の方・乳児</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/soudan/1019604.html" >}}産婦健康診査（浦安市公式）{{< /gov-link >}}

<h3>新生児聴覚スクリーニング検査の助成</h3>
<p>新生児の聴覚検査（初回）の費用を上限3,000円まで助成します（2026年6月時点）。</p>
<p><strong>対象：</strong>浦安市に住民票がある生後50日以内の乳児</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/soudan/1028728.html" >}}新生児聴覚スクリーニング（浦安市公式）{{< /gov-link >}}

<h3>はじめまして！赤ちゃん訪問（新生児訪問）</h3>
<p>助産師・保健師が赤ちゃんのいる家庭を訪問し、育児の相談に応じます。母子手帳別冊の出生連絡票を提出してください。</p>
<p><strong>対象：</strong>市内の生まれたばかりの赤ちゃんとお母さんのいる家庭</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/kenko/1000789.html" >}}赤ちゃん訪問（浦安市公式）{{< /gov-link >}}

<h3>浦安市産後ケア事業（宿泊型・日帰り型・訪問型）</h3>
<p>出産後の体の回復や育児の不安に、宿泊・日帰り・訪問の形で助産師などがサポートします。産後1年未満の母子などが対象です。</p>
{{< warn >}}利用料は施設や形態（宿泊／日帰り／訪問）によって異なります。利用を予定している施設のページで必ずご確認ください。なお、令和7年10月1日以降に出産された方は、訪問型・日帰り型の利用に事前の申請・予約が必要です。{{< /warn >}}
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/shien/1009814/index.html" >}}浦安市産後ケア事業（浦安市公式）{{< /gov-link >}}

<h3>エンゼルヘルパーの派遣</h3>
<p>出産前後や体調不良などで家事・育児が難しいとき、ヘルパーを派遣します。</p>
<p><strong>対象：</strong>周囲の支援が見込めない家庭（対象月齢・利用料などの条件あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/hoiku/1000814.html" >}}エンゼルヘルパー（浦安市公式）{{< /gov-link >}}

<h3>未熟児養育医療</h3>
<p>入院による養育が必要な未熟児の医療費を給付します。</p>
<p><strong>対象：</strong>医師が入院養育を必要と認めた未熟児</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/soudan/1000780.html" >}}未熟児養育医療（浦安市公式）{{< /gov-link >}}

<h2>そのほか・状況に応じた支援</h2>

<h3>不育症検査・治療費の助成</h3>
<p>保険適用外の不育症の検査・治療費を助成します（所得制限なし）。</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/1011499.html" >}}不育症検査・治療費助成（浦安市公式）{{< /gov-link >}}

<h3>流産・死産を経験された方への支援</h3>
<p>流産・死産・新生児死亡を経験された方へ、相談先のご案内や利用できる支援があります。おひとりで抱え込まず、ご相談ください。</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/ninshin/1041686.html" >}}流産・死産を経験された方へ（浦安市公式）{{< /gov-link >}}

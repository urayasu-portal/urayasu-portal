<p>浦安市には、子育て世帯の基本となる「児童手当」のほか、学校の費用が心配なときの就学援助、進学のための奨学支援金など、家計を支える制度があります。このページは「自分に関係しそうな支援に気づき、正しい窓口にたどり着く」ための地図です。<strong>手当の額・所得の取り扱い・申請期限などの詳細は、必ず各リンク先の浦安市公式ページでご確認ください</strong>（掲載情報は2026年6月時点のものです）。</p>

{{< note >}}
<strong>まずは児童手当の申請を（期限に注意）</strong><br>
出生や転入のあと、子育て支援課で児童手当を申請します。申請が遅れるとさかのぼって受け取れない場合があるため、原則として出生・転入の翌日から15日以内を目安に、早めに手続きしてください。
{{< /note >}}

<h2>① 子育て世帯の基本の手当</h2>

<h3>児童手当</h3>
<p>高校生年代まで（18歳に達する日以後の最初の3月31日まで）の子どもを養育している方に支給される手当です。第3子以降の多子加算などの仕組みがあります。</p>
{{< warn >}}手当の月額・多子加算の額・所得の取り扱い・支給月などは、制度改正や世帯の状況によって変わります。金額は本文では断定できないため、必ず公式ページでご自身の場合をご確認ください。{{< /warn >}}
<p><strong>対象：</strong>高校生年代までの子どもを養育している方</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/kosodate/teate/1000819.html" >}}児童手当（浦安市公式）{{< /gov-link >}}

<h2>② 小・中学校の費用が心配なとき</h2>

<h3>就学援助費制度</h3>
<p>経済的な理由で小・中学校への就学が難しい家庭に、学用品費・給食費・修学旅行費・入学前の入学準備金などを援助する制度です。申請は学校を通じて行い、お子さん1人につき1枚の申請が必要です。</p>
<p><strong>対象：</strong>経済的にお困りで、市立小・中学校に在学（入学予定を含む）のお子さんがいる家庭</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/shokai/1026105.html" >}}就学援助費制度（浦安市公式）{{< /gov-link >}}

<h2>③ 高校・大学などへの進学費用の支援</h2>

<h3>奨励金・奨学金（奨学支援金・奨学資金貸付など）</h3>
<p>経済的な理由で進学が難しい場合に、返還不要の奨学支援金や、奨学資金の貸付などがあります。募集時期が決まっているため、早めに確認しておきましょう。</p>
<p><strong>対象：</strong>経済的な理由で高校・大学・専修学校などへの進学・在学に支援が必要な方（要件・募集時期あり）</p>
{{< gov-link url="https://www.city.urayasu.lg.jp/kodomo/gakko/shogakukin/index.html" >}}奨励金・奨学金など（浦安市公式）{{< /gov-link >}}

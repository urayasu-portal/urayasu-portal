浦安で私たちが耳にする東京ディズニー旅行のトラブル──売り切れたパークチケット、弾かれたクレジットカード、夜9時の診療所探し──の多くは、数週間前なら防げたものです。これは、初めて訪れる家族に手渡したいチェックリストを、カウントダウン形式でまとめたものです。詳しい解説がある項目は各ガイド記事にリンクしています。

## 2〜3か月前

- [ ] **家族全員のパスポートを確認する。** 子どものパスポートは大人より有効期間が短く、直前に気づく定番ポイントです。
- [ ] **ホテルを予約する。** このエリアでは星の数より立地戦略が重要です。[全48ホテル比較](/travel-guide/hotels/)、[区分の仕組み](/travel-guide/hotels/types/)、[ハッピーエントリー](/travel-guide/hotels/happy-entry/)にディズニーホテルの価格差を払う価値があるかを確認。
- [ ] **チケット発売日をカレンダーに入れる。** パークチケットは**来園日の約2か月前**発売・日付とパークの指定制で、土日祝は本当に売り切れます。海外カードが公式サイトで弾かれたときの対処法まで含めて[チケットガイド](/travel-guide/disney-tickets/)へ。
- [ ] **チームラボプラネッツに行くなら予約する。** 30分で行ける[裸足のデジタルアート](/travel-guide/teamlab-planets/)も日時指定制で、週末は売り切れます。

## 1か月前

- [ ] **通信手段を用意する。** 出国前に買うeSIMなら、時差ボケの子どもを連れて空港のSIMカウンターに並ばずに済みます。舞浜エリアは主要各社とも問題なくつながります。
- [ ] **旅行保険に入り、証券を写真に撮る。** 日本の診療所は優秀で（米国基準では）安価ですが、[夜間の急病診療所は現金のみ](/travel-guide/kids-emergency/)。保険請求には領収書が必要です。
- [ ] **東京ディズニーリゾート公式アプリを入れる**。自国のストアからサインインできるか事前に確認を。当日はプレミアアクセスや待ち時間の確認に必須です。詳細は[チケットガイド](/travel-guide/disney-tickets/)。
- [ ] **スマホにSuicaを設定する**（Apple Wallet／Google Wallet）。このサイトで紹介する電車・バスは、[都心への観光ルート](/travel-guide/tokyo-sightseeing-access/)も含めてワンタップで乗れます。
- [ ] **ホテルのシャトル時刻表をスクショしておく。** シャトルのあるホテルでは[最終バスの時刻が夜の過ごし方を決めます](/travel-guide/hotels/shuttle/)。

## 1週間前

- [ ] **現金の計画を立てる。** 日本は評判より現金レスですが、小さな飲食店・ロッカー・現金のみの[急病診療所](/travel-guide/kids-emergency/)に備え、家族で2〜3万円は用意を。空港とコンビニのATMは海外カードが使えます。
- [ ] **短パンを詰める**。チームラボプラネッツに行くなら[膝まで水に入ります](/travel-guide/teamlab-planets/)（貸出はありますが自分のものが快適です）。
- [ ] **薬とアレルギーカードを用意する。** 使い慣れた子ども用解熱剤と、食物アレルギーを日本語で書いたスマホのメモ（飲食店で見せる用）を。
- [ ] **靴は今から履き慣らす。** パーク1日2万歩は普通です。初日に新品の靴を下ろすと2日目にマメで泣きます。

## 出発前日

- [ ] **オフラインマップをダウンロード**し、予約（ホテル・チケット・チームラボ）を全部スクショ。着陸直後に通信できなくても動けます。
- [ ] **初日は「軟着陸」させる。** 太平洋を越えた直後の子連れで、到着日にパークチケットを使うのはもったいない。無料で気楽な[浦安市交通公園](/travel-guide/urayasu-traffic-park/)の午後や、[舞浜ユーラシアの温泉](/travel-guide/maihama-eurasia-spa/)で整える方が、行列の中で力尽きるよりずっと良い一日になります。
- [ ] **全デバイスを充電し、電源タップを1つ。** 日本のコンセントはタイプA・100V（北米と同形状）。北米以外からの旅行者は、変換プラグ1つ＋タップで家族全員分をまかなえます。

{{< note >}}
このリストに広告は含まれていません。舞浜周辺で見かける「余裕のある家族」と「疲れ切った家族」を分けているのは、たいてい1か月前の準備です。
{{< /note >}}

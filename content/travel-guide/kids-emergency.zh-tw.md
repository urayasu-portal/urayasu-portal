晚上9點，孩子的額頭發燙，而你信任的小兒科醫師遠在幾千公里外。這一刻，迪士尼之旅不再關乎攻略——而幾乎沒有任何一篇中文迪士尼攻略能幫到你。

我們就住在東京迪士尼度假區所在的浦安市，下面是我們自己的孩子生病時也在用的資訊：夜間診所在哪、打哪些電話、如何跨越語言障礙。資訊基於2025〜2026年調查（來源：浦安市官方網頁），就診前請確認最新時間。

## 按緊急程度排列的電話

| 情況 | 電話 | 語言／時間 |
|---|---|---|
| **有生命危險——叫救護車** | **119** | 24小時。請慢慢說，報出位置（「浦安、○○飯店」） |
| **旅客熱線（中文醫療協助）** | **訪日旅客熱線 050-3816-2787** | 24小時、中·英·韓語（JNTO） |
| 要不要叫救護車？（諮詢） | #7119（043-216-3668） | 24小時、日語 |
| 兒童急病電話諮詢 | #8000（043-242-9939） | 夜間·假日、日語 |
| 浦安市健康諮詢（滯留市內的旅客也可用） | 0120-24-9250 | 24小時、免費、日語 |

**現實情況：** 本地諮詢電話大多僅限日語。作為旅客，你最有力的兩個工具是**訪日旅客熱線（中文、24小時）**和**飯店櫃檯**——舞濱地區的飯店經常處理孩子生病的情況，能幫你打電話、說明症狀、叫計程車。別猶豫叫醒他們，在日本這很正常。

---

## 在園內：先去救護室

如果孩子在**東京迪士尼樂園或迪士尼海洋園內**感到不適，請前往**救護室**（First Aid，園區地圖有標註，演藝人員會指引）。救護室有護理師常駐、免費，能幫你判斷需要就醫還是休息後繼續遊玩。必要時園方也能叫救護車。

---

## 夜間：浦安市急病診療所

閉園後的發燒、腸胃炎等突發疾病，可前往浦安市營運的**急病診療所**——內科·小兒科的應急門診。

| | |
|---|---|
| **夜間診療** | 每天 20:00〜23:00 |
| **假日白天** | 週日·假日·年末年初 10:00〜17:00（小兒科為指定日） |
| **電話** | 047-381-9999（僅診療時間內接聽、日語——建議先請飯店代打） |
| **地址** | 浦安市貓實1-2-5 健康中心1樓（[Google地圖](https://www.google.com/maps/search/浦安市健康センター+猫実一丁目2番5号)） |
| **交通** | 從舞濱地區飯店搭計程車約10〜15分鐘——參見[計程車指南](/zh-tw/travel-guide/urayasu-taxi-airport-flat-rate-guide/) |
| **付費** | **只收現金。** 處方原則上約1天份 |

就診前要知道的三件事：

1. **先打電話**（或請飯店代打）說明症狀。
2. 這裡提供的是**應急處置**——割傷、骨折、燙傷、眼睛問題會被轉介到急救醫院。拿不準嚴重程度時，用119或訪日旅客熱線。
3. **攜帶：** 現金、護照、孩子正在服用的藥，並記下症狀開始的時間（用翻譯App截圖給醫師看很有效）。

**週日·假日突然牙痛？** 同一棟樓內的假日急救牙科診療所（047-381-4749）在週日·假日·年末年初的9:00〜12:00提供應急牙科處置。

## 在地人的預防層

- **選一家館內或隔壁有便利商店的飯店**——深夜買退熱貼、飲料和兒童用品不用出樓。[親子飯店指南](/zh-tw/travel-guide/hotels/kids/)裡有清單（東京灣希爾頓飯店的24小時便利商店、舞濱夢想之門飯店樓內的NewDays等）。
- 舞濱·新浦安·浦安各站附近的**藥妝店**（認明「薬局／ドラッグストア」招牌）有兒童退燒藥、口服補液和體溫計。給藥師看翻譯App就能溝通。
- **出發前拍下旅遊保險單**，保留診所所有收據——日本的診所會開立適用於理賠的明細收據。
- **行程排太滿＋孩子疲勞＝發燒的經典配方。** 多日行程裡安排一個慢節奏的早晨，是最便宜的藥。

## 總結

1. 有生命危險 → 立即**119**。
2. 不知所措、需要中文 → **訪日旅客熱線 050-3816-2787**（24小時）或**飯店櫃檯**。
3. 夜間發燒·腸胃炎 → **浦安市急病診療所**，每晚20:00〜23:00，只收現金，先打電話。
4. 在園內 → **救護室**。
5. 預防 → 選館內有便利商店的飯店＋合理的行程節奏。

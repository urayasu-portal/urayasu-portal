晚上9点，孩子的额头发烫，而你信任的儿科医生远在几千公里外。这一刻，迪士尼之旅不再关乎攻略——而几乎没有任何一篇中文迪士尼攻略能帮到你。

我们就住在东京迪士尼度假区所在的浦安市，下面是我们自己的孩子生病时也在用的信息：夜间诊所在哪、打哪些电话、如何用中文解决问题。信息基于2025〜2026年调查（来源：浦安市官方网页），就诊前请确认最新时间。

<div class="tg-notice-info">
<i class="ti ti-alert-triangle" aria-hidden="true"></i> <strong>孩子失去意识、呼吸困难或严重受伤时，请立即拨打119（救护车）。</strong>本文帮你找到就诊渠道，不构成医疗建议。
</div>

## 按紧急程度排列的电话

| 情况 | 电话 | 语言／时间 |
|---|---|---|
| **有生命危险——叫救护车** | **119** | 24小时。请慢慢说，报出位置（「浦安、○○酒店」） |
| **游客热线（中文医疗协助）** | **访日游客热线 050-3816-2787** | 24小时、中·英·韩语（JNTO） |
| 要不要叫救护车？（咨询） | #7119（043-216-3668） | 24小时、日语 |
| 儿童急病电话咨询 | #8000（043-242-9939） | 夜间·节假日、日语 |
| 浦安市健康咨询（滞留市内的游客也可用） | 0120-24-9250 | 24小时、免费、日语 |

**现实情况：** 本地咨询电话大多仅限日语。作为游客，你最有力的两个工具是**访日游客热线（中文、24小时）**和**酒店前台**——舞滨地区的酒店经常处理孩子生病的情况，能帮你打电话、说明症状、叫出租车。别犹豫叫醒他们，在日本这很正常。

---

## 在园内：先去救护室

如果孩子在**东京迪士尼乐园或迪士尼海洋园内**感到不适，请前往**救护室**（First Aid，园区地图有标注，演职人员会指引）。救护室有护士常驻、免费，能帮你判断需要就医还是休息后继续游玩。必要时园方也能叫救护车。

---

## 夜间：浦安市急病诊疗所

闭园后的发烧、肠胃炎等突发疾病，可前往浦安市运营的**急病诊疗所**——内科·小儿科的应急门诊。

| | |
|---|---|
| **夜间诊疗** | 每天 20:00〜23:00 |
| **节假日白天** | 周日·节假日·年末年初 10:00〜17:00（小儿科为指定日） |
| **电话** | 047-381-9999（仅诊疗时间内接听、日语——建议先请酒店代打） |
| **地址** | 浦安市猫实1-2-5 健康中心1楼（[Google地图](https://www.google.com/maps/search/浦安市健康センター+猫実一丁目2番5号)） |
| **交通** | 从舞滨地区酒店打车约10〜15分钟——参见[出租车指南](/zh/travel-guide/urayasu-taxi-airport-flat-rate-guide/) |
| **付费** | **只收现金。** 处方原则上约1天量 |

就诊前要知道的三件事：

1. **先打电话**（或请酒店代打）说明症状。
2. 这里提供的是**应急处置**——割伤、骨折、烫伤、眼睛问题会被转介到急救医院。拿不准严重程度时，用119或访日游客热线。
3. **携带：** 现金、护照、孩子正在服用的药，并记下症状开始的时间（用翻译App截图给医生看很有效）。

**周日·节假日突然牙痛？** 同一栋楼内的休日急救牙科诊疗所（047-381-4749）在周日·节假日·年末年初的9:00〜12:00提供应急牙科处置。

## 本地人的预防层

- **选一家馆内或隔壁有便利店的酒店**——深夜买退热贴、饮料和儿童用品不用出楼。[亲子酒店指南](/zh/travel-guide/hotels/kids/)里有清单（东京湾希尔顿酒店的24小时便利店、舞滨梦想之门酒店楼内的NewDays等）。
- 舞滨·新浦安·浦安各站附近的**药妆店**（认准「薬局／ドラッグストア」招牌）有儿童退烧药、口服补液和体温计。给药剂师看翻译App就能沟通。
- **出发前拍下旅行保险单**，保留诊所所有收据——日本的诊所会开具适用于理赔的明细收据。
- **行程排太满＋孩子疲劳＝发烧的经典配方。** 多日行程里安排一个慢节奏的早晨，是最便宜的药。

---

## 总结

1. 有生命危险 → 立即**119**。
2. 不知所措、需要中文 → **访日游客热线 050-3816-2787**（24小时）或**酒店前台**。
3. 夜间发烧·肠胃炎 → **浦安市急病诊疗所**，每晚20:00〜23:00，只收现金，先打电话。
4. 在园内 → **救护室**。
5. 预防 → 选馆内有便利店的酒店＋合理的行程节奏。

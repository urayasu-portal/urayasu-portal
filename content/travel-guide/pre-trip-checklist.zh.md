我们在浦安听到的东京迪士尼旅行麻烦——售罄的入园日、刷不过的信用卡、晚上9点找夜间诊所——大多在几周前就可以避免。这是我们想交给每位初次到访者的清单，按倒计时排列。有详细解说的项目都附了本地指南链接。

## 出发前2〜3个月

- [ ] **检查全家每个人的护照。** 儿童护照比大人的更快到期——这是最经典的临行前发现。
- [ ] **订酒店。** 在这一带，选址策略比星级更重要：看[全48家酒店比较](/zh/travel-guide/hotels/)、[分类体系解说](/zh/travel-guide/hotels/types/)，以及对你家而言[Happy Entry](/zh/travel-guide/hotels/happy-entry/)是否值得迪士尼酒店的价格。
- [ ] **把购票日记进日历。** 入园门票在**来园日约2个月前**开卖，指定日期与园区，周末·节假日真的会售罄。你需要知道的都在[门票指南](/zh/travel-guide/disney-tickets/)里——包括官方网站刷不过海外卡时的对策。
- [ ] **想去teamLab Planets就先订。** 30分钟外的[赤脚数字艺术馆](/zh/travel-guide/teamlab-planets/)是日期时段指定制，周末同样会售罄。

## 出发前1个月

- [ ] **搞定上网。** 在家先买好eSIM，胜过带着倒时差的孩子在机场SIM柜台排队。舞滨一带主要运营商都没问题。
- [ ] **买旅行保险并拍下保单。** 以国内标准看，日本的诊所又好又不贵，但[夜间诊所只收现金](/zh/travel-guide/kids-emergency/)，保险理赔也要收据。
- [ ] **安装东京迪士尼度假区官方App**，并确认能从你所在地区登录。当天要靠它买尊享卡（DPA）、看等候时间——详见[门票指南](/zh/travel-guide/disney-tickets/)。
- [ ] **在手机里设置Suica。** 一「贴」就能坐本指南里所有的电车和巴士，包括[进东京市区的路线](/zh/travel-guide/tokyo-sightseeing-access/)。**注意：部分中国产安卓手机（如华为）无法添加移动Suica，可在机场买实体「Welcome Suica」。**
- [ ] **把酒店的班车时刻表截图。** 如果酒店有班车，[末班车时间决定你的夜晚](/zh/travel-guide/hotels/shuttle/)。

## 出发前1周

- [ ] **规划现金。** 日本比传闻中更无现金，但每家仍带上2〜3万日元，用于小餐馆、置物柜和只收现金的[急病诊所](/zh/travel-guide/kids-emergency/)。机场和便利店的ATM可用海外卡。**在检票闸机不能扫支付宝/微信二维码**（详见[市区交通指南](/zh/travel-guide/tokyo-sightseeing-access/)）。
- [ ] **带短裤**，如果行程里有teamLab Planets——你会[膝盖没入水中](/zh/travel-guide/teamlab-planets/)（有出借，但自己的更舒服）。
- [ ] **带药品和过敏卡。** 带上你信得过的儿童退烧药，以及一张写好食物过敏的日语手机备忘（给餐厅看）。
- [ ] **现在就把鞋穿松。** 入园一天2万步是常态。第一天穿新鞋，第二天就会磨出水泡。

## 出发前一天

- [ ] **下载离线地图**（舞滨·浦安），并把所有预订（酒店、门票、teamLab）截图，以防落地时没网。
- [ ] **安排「软着陆」。** 带孩子飞越太平洋后，别把到达日浪费在一张门票上。免费又轻松的[浦安市交通公园](/zh/travel-guide/urayasu-traffic-park/)半天，或早早去[舞滨欧亚温泉](/zh/travel-guide/maihama-eurasia-spa/)泡一场，都好过在队伍里崩溃。
- [ ] **把设备都充满，带一个排插。** 日本是A型插座·100V，和中国的A型插头形状相同，可直接插；手机·相机充电器多为宽电压（100–240V）可直接用，但仅220V的电器（如吹风机）不适用。带一个排插，全家共用一个转换头即可。

{{< note >}}
这份清单没有任何广告。它只是把我们在舞滨见到的「从容的家庭」和「手忙脚乱的家庭」区分开的那些步骤——迪士尼旅行中的大多数「倒霉」，其实一个月前就已注定。
{{< /note >}}

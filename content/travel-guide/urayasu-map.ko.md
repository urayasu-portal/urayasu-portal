파크 밖의 “곤란한 순간”을 이 지도 한 장으로. 숙박 중인 호텔을 선택하고, 니즈(기저귀를 사고 싶다／심야에 먹고 싶다／해외카드로 현금을 뽑고 싶다 등)나 매장 종류로 좁히면, 가장 가까운 매장과 도보 시간 기준이 지도에 표시됩니다.

{{< facility-map >}}

## 사용 방법

- **숙박 호텔 선택** … 지도가 해당 호텔을 중심으로 이동하며 ★ 표시와 반경 400m 기준 원을 표시합니다. 각 매장 팝업에 “호텔에서 도보 ◯분”이 나옵니다.
- **니즈별(코럴색 버튼)** … “기저귀·분유” “도시락·간식” “심야 식사” “아침 식사” “현금(해외카드)” “세탁” “상비약” “갈아입을 옷” “일본 음식” “면세” “짐 배송” 중에서 선택하면 용도에 맞는 매장만 표시합니다.
- **매장 종류(아래 버튼)** … 편의점·슈퍼마켓·드러그스토어·음식·카페·100엔숍·가전·의류·세탁소·우체국·상업시설(복수 선택 가능).
- **지금 영업 중** … 현재 시각에 영업 중인 매장만 표시합니다(영업시간 기준 간이 판정).

영업시간·도보 시간은 참고용이며 심야·이른 아침·공휴일에는 달라질 수 있습니다. 최신 정보는 각 매장 공식 페이지(팝업의 “매장 검색”)에서 확인하세요.

我們在浦安聽到的東京迪士尼旅行麻煩——售罄的入園日、刷不過的信用卡、晚上9點找夜間診所——大多在幾週前就可以避免。這是我們想交給每位初次到訪者的清單，按倒數排列。有詳細解說的項目都附了在地指南連結。

## 出發前2〜3個月

- [ ] **檢查全家每個人的護照。** 兒童護照比大人的更快到期——這是最經典的臨行前發現。
- [ ] **訂飯店。** 在這一帶，選址策略比星級更重要：看[全48家飯店比較](/zh-tw/travel-guide/hotels/)、[分類體系解說](/zh-tw/travel-guide/hotels/types/)，以及對你家而言[Happy Entry](/zh-tw/travel-guide/hotels/happy-entry/)是否值得迪士尼飯店的價格。
- [ ] **把購票日記進行事曆。** 入園門票在**來園日約2個月前**開賣，指定日期與園區，週末·連假真的會售罄。你需要知道的都在[門票指南](/zh-tw/travel-guide/disney-tickets/)裡——包括官方網站刷不過海外卡時的對策。
- [ ] **想去teamLab Planets就先訂。** 30分鐘外的[赤腳數位藝術館](/zh-tw/travel-guide/teamlab-planets/)是日期時段指定制，週末同樣會售罄。

## 出發前1個月

- [ ] **搞定上網。** 在家先買好eSIM，勝過帶著調時差的孩子在機場SIM櫃檯排隊。舞濱一帶主要電信商都沒問題。
- [ ] **買旅遊保險並拍下保單。** 以台灣標準看，日本的診所又好又不貴，但[夜間診所只收現金](/zh-tw/travel-guide/kids-emergency/)，保險理賠也要收據。
- [ ] **安裝東京迪士尼度假區官方App**，並確認能從你所在地區登入。當天要靠它買尊享卡（DPA）、看等候時間——詳見[門票指南](/zh-tw/travel-guide/disney-tickets/)。
- [ ] **在手機裡設定Suica。** 一「嗶」就能搭本指南裡所有的電車和巴士，包括[進東京市區的路線](/zh-tw/travel-guide/tokyo-sightseeing-access/)。
- [ ] **把飯店的接駁車時刻表截圖。** 如果飯店有接駁車，[末班車時間決定你的夜晚](/zh-tw/travel-guide/hotels/shuttle/)。

## 出發前1週

- [ ] **規劃現金。** 日本比傳聞中更無現金，但每家仍帶上2〜3萬日圓，用於小餐館、置物櫃和只收現金的[急病診所](/zh-tw/travel-guide/kids-emergency/)。機場和便利商店的ATM可用海外卡。
- [ ] **帶短褲**，如果行程裡有teamLab Planets——你會[膝蓋沒入水中](/zh-tw/travel-guide/teamlab-planets/)（有出借，但自己的更舒服）。
- [ ] **帶藥品和過敏卡。** 帶上你信得過的兒童退燒藥，以及一張寫好食物過敏的日語手機備忘（給餐廳看）。
- [ ] **現在就把鞋穿鬆。** 入園一天2萬步是常態。第一天穿新鞋，第二天就會磨出水泡。

## 出發前一天

- [ ] **下載離線地圖**（舞濱·浦安），並把所有預訂（飯店、門票、teamLab）截圖，以防落地時沒網。
- [ ] **安排「軟著陸」。** 帶孩子飛越太平洋後，別把抵達日浪費在一張門票上。免費又輕鬆的[浦安市交通公園](/zh-tw/travel-guide/urayasu-traffic-park/)半天，或早早去[舞濱歐亞溫泉](/zh-tw/travel-guide/maihama-eurasia-spa/)泡一場，都好過在隊伍裡崩潰。
- [ ] **把設備都充滿，帶一個延長線。** 日本是A型插座·100V。台灣也是A型、電壓相近（台110V／日100V），幾乎不用轉換頭；手機·相機充電器可直接用。帶一條延長線，全家共用一個插座即可。

{{< note >}}
這份清單沒有任何廣告。它只是把我們在舞濱見到的「從容的家庭」和「手忙腳亂的家庭」區分開的那些步驟——迪士尼旅行中的大多數「倒楣」，其實一個月前就已注定。
{{< /note >}}

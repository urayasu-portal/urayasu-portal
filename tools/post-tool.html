📝 浦安ぽーたる 記事投稿ツール

AIが生成した .md ファイル内容を貼り付けて、GitHubに直接投稿します

⚙️ GitHub トークン設定

GitHub → Settings → Developer settings → Personal access tokens (Fine-grained) で発行。
権限：Contents: Read and write のみ必要。設定はこのブラウザにのみ保存されます。

✅ トークンが保存されています
📋 .md ファイル内容 フロントマター＋本文をそのまま貼り付け

記事化プロンプトの「■ .mdファイル内容（フロントマター＋本文HTML）」ブロックをそのまま貼り付けてください。

チェックなしの場合はカテゴリのデフォルト画像が使われます。

📂 クリックまたは画像をドラッグ＆ドロップ

英数字・ハイフン・アンダースコアのみ。拡張子（.jpg / .png / .webp）必須。

📷 記事内に画像を追加する（任意）

📂 クリックまたは画像をドラッグ＆ドロップ（複数可）

アップロード後、生成された <img> タグを本文テキストエリアの任意の位置に挿入できます。

📄 ファイル名

形式：YYYYMMDD-slug.md　上の内容を貼り付けてから「自動入力」を押すと date と slug から自動生成されます。
